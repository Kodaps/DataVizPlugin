newCircleSegment (data)
create a circle segment

Parameters:

  • data table a table with : radius, startangle, endangle, (inner_raidus)

Returns:

    a polygon DisplayObject anchored in the center of the circle

Usage:

    local segment = dataviz.newCircleSegment({
  start_angle = 0,
  end_angle = 60,
  radius = radius,
  stroke = 5,
  inner_radius = radius*.5,
  color = "#222F",
  strokeColor = "#0FF",
  strokeWidth = 1
)
